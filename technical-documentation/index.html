Introduction

Python is a high-level, interpreted programming language known for its readability and versatility.

It is widely used in web development, automation, data science, and more.

print("Hello, World!")
  • Easy syntax
  • Cross-platform
  • Open source
Installation

You can install Python from the official website or using a package manager.

Make sure to add Python to your PATH.

sudo apt install python3
Variables

Variables store data values. Python is dynamically typed.

You don't need to declare types explicitly.

x = 10 name = "Anagha"
Functions

Functions are defined using the def keyword.

You can pass arguments and return values.

def greet(name): return "Hello " + name
Control Flow

Python supports conditional and loop statements.

Use if-elif-else for conditions and for/while loops for iteration.

if x > 0: print("Positive")
